# دليل الاستخدام بالعربية — reading_reels
### أداة توليد فيديوهات قراءة إنجليزية بنمط تيك توك (تعمل مباشرة على ويندوز 10)

هذه الأداة تُنشئ تلقائيًا فيديو عمودي 1080×1920 مدته حوالي دقيقتين:
- يبدأ بعبارتين منطوقتين: **"Let's practice English"** و **"Education is the key"**
- ثم يعرض **فقرة قراءة كاملة ومترابطة** (قصة قصيرة على موضوع واحد) بكلمات مناسبة لمستوى المتعلّم
- النص **يتحرّك ببطء للأعلى** ليبقى قابلًا للقراءة حتى آخر ثانية (لزيادة نسبة الإكمال)
- خلفية مختلفة لكل فيديو + خيار رفع تلقائي إلى تيك توك

> ✅ **تعمل فورًا بدون أي مفاتيح API** (تستعمل محرك محتوى داخلي + خلفيات متدرّجة اللون).
> أضف المفاتيح لاحقًا لتحسين الجودة. كل شيء مجاني.

---

## 1) التشغيل على ويندوز 10 — خطوة بخطوة

**المتطلب الوحيد:** تثبيت **Python 3.10+** من python.org (أثناء التثبيت فعّل خيار
"Add Python to PATH"). لا حاجة لتثبيت ffmpeg — الأداة تجلبه تلقائيًا.

1. فك ضغط الملف `reading_reels.zip` في أي مكان (مثلاً سطح المكتب).
2. افتح المجلد `reading_reels`.
3. **انقر نقرًا مزدوجًا على `setup.bat`** — سيُنشئ بيئة افتراضية ويثبّت كل المكتبات
   (قد يستغرق بضع دقائق في أول مرة).
4. بعد انتهائه، **انقر نقرًا مزدوجًا على `run.bat`** — ستظهر قائمة:
   - اختر **1** لصنع فيديو واحد (اختر المستوى A1/A2/B1/B2/C1).
   - اختر **2** لصنع فيديو ورفعه تلقائيًا إلى تيك توك.
   - اختر **3** لصنع 5 فيديوهات دفعة واحدة.
   - اختر **4** لتوليد محتوى لكل الحسابات.
   - اختر **6** لفحص البيئة (doctor).
   - اختر **7** لفتح مجلد النتائج.
5. تجد الفيديوهات الجاهزة داخل: `outputs\jobs\<التاريخ>_<الاسم>\`
   - `*.mp4` ← الفيديو الجاهز للرفع
   - `caption.txt` ← الوصف + الهاشتاقات (انسخها عند الرفع)
   - `paragraph.txt` ← الفقرة الكاملة كنص
   - `script.json` / `metadata.json` / `timing.json`

**أوامر بديلة من سطر الأوامر (PowerShell):**
```powershell
cd reading_reels
py -3 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m app.main doctor
python -m app.main one --level B1
```

## 2) إضافة مفاتيح API — خطوة بخطوة (كلها مجانية واختيارية)

أنشئ ملف `.env` بنسخ الملف `.env.example`، ثم الصق المفاتيح بعد علامة `=`.
(لا تشارك هذا الملف مع أحد.)

### أ) NVIDIA NIM — لتوليد فقرات قراءة بالذكاء الاصطناعي (موصى به)
1. ادخل إلى **https://build.nvidia.com** وسجّل دخول مجاني.
2. افتح أي نموذج مثل **Llama 3.1 70B Instruct**.
3. اضغط **Get API Key** ← **Generate Key**.
4. انسخ المفتاح (يبدأ بـ `nvapi-`) والصقه في `.env`:
   ```
   NVIDIA_API_KEY=nvapi-xxxxxxxxxxxxxxxxxxxxxxxx
   ```

### ب) Pexels — خلفيات صور/فيديو حقيقية مجانية
1. ادخل **https://www.pexels.com/api/** وسجّل دخول.
2. انسخ المفتاح من لوحة التحكم والصقه:
   ```
   PEXELS_API_KEY=مفتاحك
   ```

### ج) Pixabay — مصدر خلفيات إضافي (اختياري)
1. ادخل **https://pixabay.com/api/docs/** وسجّل دخول.
2. انسخ المفتاح والصقه:
   ```
   PIXABAY_API_KEY=مفتاحك
   ```

تحقّق بكتابة: `python -m app.main doctor` — سيظهر أي المفاتيح مفعّلة.

## 3) إضافة حسابات تيك توك — خطوة بخطوة

عدّل الملف `config/accounts.yaml` وأضف كتلة لكل حساب (حساب واحد = هوية تعليمية واحدة):
```yaml
accounts:
  - id: "read_a2_travel"
    display_name: "Easy Travel English"
    level: "A2"                       # A1/A2/B1/B2/C1
    topic_families: ["travel", "daily_life"]
    audience: "elementary learners"
    style: "calm, cinematic"
    posting:
      per_day: 2
      best_times: ["08:00", "19:00"]
```
ثم شغّل: `python -m app.main accounts` لتوليد محتوى كل حساب.

## 4) الرفع التلقائي إلى تيك توك — خطوة بخطوة

هناك طريقتان (التفاصيل الكاملة في `docs/UPLOAD_TIKTOK.md`):

### الطريقة (web) — الأسهل لجهازك (تعمل عبر متصفح Chrome بحسابك)
1. تأكد من تثبيت **Google Chrome** على ويندوز.
2. في `config/config.yaml` اجعل:
   ```yaml
   upload:
     enabled: true
     method: "web"
   ```
3. شغّل أول مرة: `python -m app.main one --level B1 --auto-upload`
4. ستفتح نافذة Chrome. **سجّل دخولك إلى تيك توك مرة واحدة**، ثم ارجع إلى نافذة
   الأوامر واضغط **Enter**. سيتم رفع الفيديو وكتابة الوصف والنقر على Post تلقائيًا.
5. بعد ذلك يبقى تسجيل دخولك محفوظًا، فكل رفع لاحق يكون تلقائيًا بالكامل.

> ملاحظة مهمة: هذه الطريقة تتحكّم بجلستك أنت. واجهة تيك توك تتغيّر أحيانًا، لذلك يمكن
> تعديل المُحدّدات (selectors) من `config/config.yaml` عند الحاجة. استخدمها بوتيرة
> بشرية معقولة للحفاظ على حسابك.

### الطريقة (api) — الرسمية (تتطلب موافقة تطبيق مطوّر من تيك توك)
- تحتاج Access Token بصلاحية `video.publish`. ضعه في `.env`:
  ```
  TIKTOK_ACCESS_TOKEN=...
  ```
- واجعل في `config/config.yaml`: `upload.method: "api"`.

## 5) حل المشكلات الشائعة
- **لا يوجد صوت:** gTTS يحتاج إنترنت؛ أو اجعل `audio.intro_tts: false`.
- **الذكاء الاصطناعي يستعمل البديل المحلي دائمًا:** لم تضع `NVIDIA_API_KEY` أو غير صحيح.
- **الخلفيات دائمًا متدرّجة الألوان:** لم تضع `PEXELS_API_KEY`/`PIXABAY_API_KEY`.
- **الرفع لا يعمل:** ثبّت `pip install selenium` و Google Chrome، وتأكد أنك سجّلت الدخول.
- **العرض بطيء:** اجعل `video.preset: veryfast` في `config/config.yaml`.

للمزيد راجع المجلد `docs/` (كل الأدلة بالإنجليزية بالتفصيل).
